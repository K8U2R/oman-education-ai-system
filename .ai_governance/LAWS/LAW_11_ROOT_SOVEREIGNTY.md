# القانون 11: التشغيل المركزي (Root Sovereignty)

## التعريف (Definition)

جذر المشروع هو مركز القيادة الوحيد. يُحظر تشغيل أوامر داخل المجلدات الفرعية يدوياً.

## السبب التقني (The Rationale)

1. **سلامة البيئة:** ضمان تحميل ملفات `.env` وتوحيد المسارات.
2. **منع التضارب:** إدارة العمليات المتزامنة (Backend + Frontend) بشكل صحيح.

## القائمة البيضاء (المسموح ✅)

- `npm run dev` (من الجذر).
- `npm run db:migrate` (من الجذر).

## القائمة السوداء (المحظور ❌)

```bash
cd backend && npm start
```

## آلية التنفيذ (Enforcement)

- توثيق الأوامر في `README.md`.
